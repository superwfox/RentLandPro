# RentLandPro

基于区块的 Minecraft 领地租赁系统，支持 QQ 群组通知与经验货币。

## 功能特性

### 领地管理
- **区块级保护** — 以 16×16 区块为单位圈地，支持多区块组合
- **租期系统** — 按天计费，使用玩家经验等级作为货币
- **权限分级** — operators（管理员）/ members（普通成员）/ visitors（游客）

### GUI 菜单
- **领地主页** — 查看所有拥有或可管理的领地
- **领地详情** — 修改名称、租期、标志、成员、传送点
- **功能设置** — 雪球拖拽式开关，控制游客的破坏/放置/交互/爆炸权限
- **成员管理** — 在线玩家头颅列表，分页浏览，一键添加

### QQ 通知（OneBot）
- 领地创建时群发通知 + @地主
- 游客进入领地时私聊地主（2分钟冷却防骚扰）
- 租期剩余 5/2/0 天时私聊提醒

## 使用方法

### 创建领地
1. 手持 **书与笔**
2. 右键点击地面方块
3. 重复步骤 2 扩展领地范围
4. 输入 `/land confirm` 完成创建

### 管理领地
1. 输入 `/land home` 打开领地主页
2. 点击领地图标进入详情菜单
3. 选择要修改的选项

### 功能开关
| 功能 | 说明 |
|------|------|
| 破坏 | 允许游客破坏方块 |
| 放置 | 允许游客放置方块 |
| 交互 | 允许游客与方块交互（箱子、门等） |
| 爆炸 | 允许爆炸破坏领地内方块 |

雪球在 **第二行** = 开启，**第三行** = 关闭

### 删除领地
将租期修改为 `0` 即可删除领地，经验会按剩余天数退还。

## 配置文件

`plugins/RentLandPro/config.yml`:
```yaml
# QQ群号（用于通知）
QQGroupId: "123456789"

# 每区块每天的租金（经验等级）
ChunkPricePerDay: 5
```

## 数据文件

| 文件 | 内容 |
|------|------|
| `landInfo.dat` | 领地基本信息（名称、地主、租期、标志、区块列表） |
| `permission.dat` | 领地成员关系（operators / members） |
| `functions.dat` | 领地功能开关（bitfield） |

## 命令

| 命令 | 说明 |
|------|------|
| `/land home` | 打开领地主页 |
| `/land confirm` | 确认领地创建/扩展 |
| `/land cancel` | 取消当前编辑 |

## 权限节点

| 节点 | 说明 |
|------|------|
| `rentland.create` | 创建领地 |
| `rentland.admin` | 管理员权限 |

## 依赖

- **服务端**: Paper/Spigot 1.20+
- **前置插件**: 无
- **外部服务**: OneBot（可选，用于 QQ 通知）

## 安装

1. 将 `RentLandPro.jar` 放入 `plugins/` 目录
2. 启动服务器生成配置文件
3. 编辑 `config.yml` 设置 QQ 群号和租金
4. 重启服务器或 `/reload`
