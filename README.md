# RentLandPro

基于区块的 Minecraft 领地租赁插件，使用玩家**经验等级**作为货币，并通过 OneBot 与 QQ 群联动。

---

## 玩家使用

### 创建领地
1. 手持 **空地图 (`MAP`)** 站到目标区块内
2. **右键** 方块 → 把当前区块加入待选范围；插件会发给你一张专属编辑地图
3. 继续 **右键** 添加相邻区块，**左键** 取消已选区块
4. 也可以 **丢出地图** 框选两个对角，一次性添加矩形区域内的全部空闲区块
5. 输入 `/land confirm [天数]` 完成创建（天数省略默认 1 天）
6. 输入 `/land cancel` 取消

> 5 分钟内未确认会自动取消。

### 修改与管理
- `/land` 直接打开**领地主页**，列出你作为地主或成员的全部领地
- 在主页**左键**领地图标 → 传送到该领地
- 在主页**右键**领地图标（仅限地主） → 打开**领地详情菜单**

详情菜单中可以：
- 修改领地**名称**（聊天栏输入）
- 修改**租期**（聊天栏输入新的天数；输入 `0` 删除领地）
- 修改领地**标志方块**（手持物品后点击）
- 管理**成员**（在线玩家头颅，分页添加/移除）
- 设置游客**功能开关**（破坏 / 放置 / 交互 / 爆炸）
- 设置**传送点**（必须在距领地中心 6 区块以内）
- **重塑范围**（重新进入区块编辑模式，按差价多退少补）

### 租期与计费
- 后端按**小时**倒计时，前端显示按**天**（保留两位小数）
- 一天的价格 = `面积 ^ 0.8891`（向下取整，单位为经验等级）
- **新建** / **续期**：花费 = 一天价格 × 天数
- **重塑范围**：按剩余小时比例补/退新旧面积差价
- **修改租期为 0（删除领地）**：不退整天部分，只按剩余「不足一天的零头小时」按比例退还
- **租期到期**：自动删除领地，向地主 QQ 发送通知

### 过期提醒
剩余 **120 小时（5 天）** 和 **48 小时（2 天）** 时，会向地主 QQ 私聊提醒一次。

### QQ 群命令
| 命令 | 说明 |
|------|------|
| `我的领地` | 列出自己作为地主/成员的领地 |
| `领地列表` | 列出全服所有领地（最多 20 条） |
| `查询领地 <名称>` | 查询单个领地详情 |

群成员需先通过 Courier 之类的工具完成 **QQ ↔ 游戏 ID** 绑定，才能识别身份。

## 管理员

### 游戏内命令
| 命令 | 说明 |
|------|------|
| `/land return` | 站在某个领地的任意区块内，强制收回该领地。**不退款**，但会在聊天栏与 QQ 群打印 `领地价值 ： <一天价格>`，并 @ 地主告知 |

### QQ 私聊命令（仅限 `OpQQ` 列表中的 QQ）
| 命令 | 说明 |
|------|------|
| `领地总览` | 返回领地总数、总区块数、所有管理员与普通成员名单（格式 `name(qqNum)`） |

> 续费 / 删除领地不再提供 QQ 命令入口；如需删除请使用游戏内 `/land return` 或让地主自行将租期改为 0。

## 服主须知

### 配置文件
`plugins/RentLandPro/config.yml`：
```yaml
QQGroupId: "1076566686"   # 用于通知的 QQ 群号，必须是字符串
ChunkPricePerDay: 1       # 当前未参与计费，保留字段
OpQQ:                     # 可使用 QQ 私聊管理命令的管理员 QQ 列表
  - "2054565750"
  - "2963502563"
```

### 计费公式
一天的价格固定为 `floor(面积 ^ 0.8891)`，**写死在代码里**，目前不可通过配置修改。如果需要调整计费曲线，需要改 `ConfigManager.calculateLandPrice`。

举例：
| 面积（区块） | 一天的价格（经验等级） |
|---|---|
| 1 | 1 |
| 4 | 3 |
| 9 | 7 |
| 16 | 11 |
| 25 | 16 |

### 数据文件
插件运行时会在 `plugins/RentLandPro/` 下产生：

| 文件 | 内容 |
|---|---|
| `landInfo.dat` | 领地基本信息（名称、地主、**租期(小时)**、标志、区块、传送点） |
| `permission.dat` | 领地成员关系（管理员 / 普通成员） |
| `functions.dat` | 游客功能开关位掩码 |

> ⚠️ **租期字段单位为小时**。如果你从更早的「按天存储」版本升级，旧文件里的整数会被**直接当作小时**读取（例如旧的「30 天」会变成「30 小时」）。**升级前请手动备份并把旧值乘以 24**，本插件不做自动迁移。

### OneBot / QQ 联动
- 插件通过 WebSocket 连 OneBot 实现 QQ 收发，需要自行部署兼容 OneBot v11 的机器人后端（go-cqhttp / NapCat / Lagrange 等）
- `QQGroupId` 必须是机器人已加入的群
- `OpQQ` 列表用于鉴权 QQ 私聊命令，请只填可信管理员
- 玩家与 QQ 的绑定关系由外部 Courier CSV 提供，本插件只读取

### 主世界限制
**只有主世界**允许创建和重塑领地范围。其他维度内尝试编辑会被阻止，但已有领地的保护规则在所有维度均生效。

### 传送点限制
传送点必须在距离领地"起始区块"6 区块以内（曼哈顿距离），防止地主把传送点设到领地外。

### 行为定时器
- **租期倒计时**：每小时 tick 一次，每次 `-1`，到 0 自动删除领地
- **5 分钟编辑超时**：玩家进入区块编辑模式后 5 分钟未 confirm/cancel，会自动取消并清理状态

### 经济与回退策略
- 所有花费 / 退款都在玩家的**经验等级**上加减，不接其它经济插件
- 玩家经验不足时操作会被拒绝，不会出现负等级
- 过期自动删除**不退款**；管理员 `/land return` **不退款**
- 仅有「修改租期为 0」会按未满一天的零头小时按比例退款
